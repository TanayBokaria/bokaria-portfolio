One-time setup (5–10 minutes):
1. console.cloud.google.com → New Project → Enable Google Sheets API
2. APIs & Services → Credentials → Create OAuth 2.0 Client ID (Web Application)
3. Add your hosted URL to Authorised JavaScript Origins
4. Create a Google Sheet · Add tab named pms_aif_values
   Headers: id | fund | investor | investedToDate | currentValue | asOfDate | updatedBy | notes
5. Share the sheet with your accountant's Google account (Viewer is enough)
